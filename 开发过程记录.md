# 用脚手架生成vue项目
```bash
npm install -g @vue/cli
# OR
yarn global add @vue/cli
```
- 可以使用`vue --version`查看版本号
- 创建一个心得项目`vue create kamen-ui`
***
```
之后就可以用下面得代码运行项目了
```bash
cd kamen-ui
yarn serve
# OR
npm run serve
```
